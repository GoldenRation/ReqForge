# 用户管理系统 PRD

## 项目背景
构建一个通用的用户管理系统，支持用户注册、登录、个人信息管理等功能。

## 功能需求

### 1. 用户注册
- 用户通过邮箱和密码注册账号
- 密码长度 8-20 位，需包含字母和数字
- 邮箱需验证格式，不可重复注册
- 注册成功后发送验证邮件

### 2. 用户登录
- 使用邮箱 + 密码登录
- 登录成功后返回 JWT token
- 密码错误 5 次后锁定账号 30 分钟
- 支持"记住我"功能（token 有效期 7 天）

### 3. 个人信息管理
- 查看个人信息（昵称、头像、手机号等）
- 修改个人资料（昵称、头像、手机号）
- 修改密码（需验证旧密码）
- 手机号需验证格式

### 4. 管理员功能
- 管理员可查看用户列表（支持分页和搜索）
- 管理员可禁用/启用用户账号
- 管理员可查看用户统计（注册趋势、活跃度等）

## 非功能需求
- 所有 API 需做鉴权（除注册和登录）
- 接口响应时间 < 200ms (P95)
- 密码使用 bcrypt 加密存储
- 支持水平扩展
